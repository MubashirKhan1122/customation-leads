04 — Email Outreach
Cold Email Campaign System
Personalized cold emails based on actual website issues found. Automated follow-up sequences with tracking.
✉️

Personalized Emails

Each email mentions the specific score and top 3 issues found on their website. Much higher reply rate than generic cold emails.

🔄

Auto Follow-ups

3-step automated sequence: Follow-up #1 (Day 3), Follow-up #2 (Day 7), Break-up email (Day 14). Stops if they reply.

📈

Email Tracking

Invisible tracking pixel detects email opens and link clicks. Know exactly who read your email.

📝

Template Editor

4 default templates + create custom ones. Live HTML preview. Variables like {{business_name}}, {{score}}, {{issues}}.

⏱️

Rate Limiting

5-second delay between emails to avoid spam filters. Configurable delay in settings.

🔁

Auto Retry

Failed emails are automatically retried within 24 hours. Daily cron job processes everything at 9 AM.

Email Template Variables

  • ✓ {{business_name}} — Business name
  • ✓ {{first_name}} — Contact first name
  • ✓ {{score}} — Website score (0-100)
  • ✓ {{score_label}} — "critical" / "needs improvement"
  • ✓ {{issues_html}} — Top 3 issues as HTML list
  • ✓ {{issue_count}} — Total issues found
  • ✓ {{sender_name}} — Your name
  • ✓ {{company_name}} — Your company

Built-in CRM Features
Organize, filter, tag, and manage all your leads in one place.
📋

Track leads through stages: New → Contacted → Replied → Converted → Ignored. Bulk status changes for multiple leads at once.

Automatically checks domain-level duplicates before saving. No more repeated leads in your database.

08 — Security & Compliance
Security Features
🔒

Route Protection

Server-side middleware enforces authentication on all protected routes. Unauthenticated users are redirected to login. API routes return 401.

🍪

Secure Sessions

HttpOnly cookies with 7-day expiration. Secure flag in production. SameSite=Lax to prevent CSRF.

📧

CAN-SPAM Compliant

Every email includes an unsubscribe link. Unsubscribing marks lead as ignored and cancels all pending follow-ups.

🔑

Cron Security

Vercel cron endpoint validates CRON_SECRET header. Prevents unauthorized trigger of email automation.
